Generative AI 활용

생성형 AI를 “초안 생성 + 검증 + 문서화” 도구로 활용했습니다.

적용 방식

  • 요구사항을 입력/출력/예외로 구조화 → 구현 난이도 감소
  • 코드 초안 생성 → 유니티 규칙(컴포넌트/직렬화/성능)에 맞춰 리팩토링
  • 에러 로그 기반으로 재현 조건/원인 후보를 체크리스트화
  • 발표자료/문서는 초안 생성 후 근거(영상/스크린샷/링크)로 보강

품질 규칙

  • AI 결과는 “정답”이 아니라 “초안”으로 사용
  • 프로젝트 구조/네이밍/의존성은 사람이 통제
  • Play Mode 동작 확인 + 로그 기반 검증
  • 작게 만들고(프로토타입), 빠르게 개선(반복)
